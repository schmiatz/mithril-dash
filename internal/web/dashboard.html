Roadmap
This dashboard is a fully external observer: it tails mithril.log and replay_timings.jsonl, scrapes mithril's Prometheus exporter, and polls mithril_state.json and mithril's own JSON-RPC — no changes to mithril itself. Certificate/fork-choice/dual-watermark detail (exact notarization vs. finalization state, per-slot vote tallies) lives only in consensus.Engine.Snapshot() in-process today; exposing that as a small JSON debug endpoint in mithril would let a future version replace log-line parsing with exact state.
